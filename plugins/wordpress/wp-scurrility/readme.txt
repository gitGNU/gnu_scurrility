Filters profanity from comments using scurrility.

== Description ==

Many web applications allow users to submit content that is displayed to 
other users and/or the general public. For example, wordpress displays 
comments submitted by blog readers. For some wordpress installations, 
the blog owner may wish to filter out profanity.

This plugin leverages the free as in freedom network service
<a href="https://savannah.nongnu.org/projects/scurrility/">scurrility</a>
to provide a vulgarity filtering solution.

== Installation ==

Instructions for installing the wp-scurrility plugin.

1. Download and extract the wp-scurrility plugin tar.gz file.
2. Upload the wp-scurrility folder to the `/wp-content/plugins/` directory
3. Activate the Plugin via the 'Plugins' menu in WordPress.
